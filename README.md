# News Feed Application
React app that displays live news headlines via NewsAPI.
## Setup
1.	Clone the repo
git clone https://github.com/beankook/sdc-news-report
2. Install dependencies
   npm install
3. Add your API key in App.js
4. Run the app
   npm start
Features
- Category wise news
- Search by keyword
- Responsive design
- Light and dark mode
- Bookmarking
- Axios API integration
